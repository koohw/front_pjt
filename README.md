# Vue install
```
# vue 폴더로 이동
cd vue-project
```
```
# 필수 라이브러리 설치
npm install
npm axios
npm install bootstrap-vue-3
```
```
# 실행
npm run dev
```

# Django
```
# django 폴더로 이동
cd django-pjt
```
```
# 가상환경 설정
python -m venv/venv
source venv/Script/activate
```
``` 
# 필수 라이브러리 설치
pip install -r requirements.txt
```
```
# DB migrate
python manage.py migrate
python manage.py loaddata articles.json movies.json comments.json accounts.json
```
```
# 실행
python manage.py runserver
```
